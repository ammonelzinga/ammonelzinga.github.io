Egyptian Hieroglyph Recognizer

View GitHub Project

Status: About 25% done.

This project uses deep learning to read images of Egyptian hieroglyphs. The goal is to identify each symbol by Gardiner code, infer reading direction when needed, and eventually produce likely English interpretations from glyph sequences.

Current Model Plan

  • Glyph localization: YOLO object detection to find glyph boundaries in an image.
  • Glyph classification: ResNet-18 model to predict the most likely Gardiner sign for each detected glyph.
  • Direction detection: Binary ResNet-18 model to determine left/right orientation for direction-sensitive signs.
  • Sequence interpretation: Planned sequence/transformer-style model for probable English outputs.

Milestones Completed

  • Recognize the Gardiner sign based on a picture of a hieroglyph.
  • Recognize the correct direction when applicable.

Milestones Remaining

  • Train and evaluate the YOLO detector on labeled bounding-box data.
  • Expand training data and improve robustness across writing styles and image quality.
  • Combine detection + classification + direction into a single end-to-end pipeline.
  • Build the translation stage for glyph sequence to likely English meanings.
  • Define and track final performance targets (detection overlap and full-pipeline accuracy).

Data and Evaluation Snapshot

The training data comes from open hieroglyph datasets (including sources from GitHub and Hugging Face), with additional manual box labeling planned for a subset. Model training uses a train/validation split, weighted cross-entropy for class imbalance, and Adam optimization.
