Took the source code for the open source classic game GlTron and added several exciting new features and game modes. This was my first .exe Windows application!

New Features Added

  • Speed Boost Usage Indicator - Visual feedback for power-up usage
  • Scoring for Crashing Other Players - Competitive scoring system
  • 6 Players Support - Expanded from the original player limit
  • Classic Snake Mode - Trails are finite and grow after eating bugs
  • Team Mode - Revenge-style game where eliminating an opponent makes an eliminated ally rejoin the game

About GlTron

GlTron is a classic 3D lightcycle racing game inspired by the movie TRON. Players control a lightcycle that leaves a solid trail behind it, and the objective is to force opponents to crash into these trails while avoiding them yourself.

Project Highlights

  • Modified existing C/C++ codebase
  • Implemented new game mechanics and modes
  • Built and compiled Windows executable
  • My first experience creating a distributable .exe application
