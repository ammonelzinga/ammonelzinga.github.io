Project Description

This chess implementation was developed as a school project. It featured live multiplayer functionality using Java Sockets, and a server-side component that managed game sessions using a MySQL database on my local machine. Feel free to review the code and test it out locally by going to the Github shown below. Or, scroll down to enjoy a friendly chess game for yourself. Play against yourself or a friend.
