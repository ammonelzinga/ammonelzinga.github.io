Project Description

Back before Marshallese was available on Google Translate, I decided to create my own (extremely limited) translation tool. The Marshallese Translate website was my first website ever developed. It featured a primitive English to Marshallese translator built from scratch. Other features included flash cards, account management, live chat with current users, and adding friends to see their progress with the flash cards. It was formerly hosted on AWS EC2, and used a MongoDB database. However, the site was removed and all server features including chat have been disabled. The application was built with HTML, CSS, JavaScript, and Node.js.
