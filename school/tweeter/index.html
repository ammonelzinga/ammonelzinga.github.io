View on GitHub

A school project to learn software architecture. I was tasked to improve the frontend of a simple social media type site and then build the backend. This project demonstrates modern cloud architecture patterns and scalable system design.

Key Features

  • Frontend improvements to enhance user experience
  • Complete backend implementation from scratch
  • Implemented MVP (Model-View-Presenter) architectural pattern
  • Double queue system for efficient horizontal scaling
  • Fully cloud-based infrastructure using AWS services

Technologies Used

  • AWS Lambda - Serverless compute for backend logic
  • AWS SQS - Message queuing for asynchronous processing
  • AWS S3 - Object storage for user content
  • AWS DynamoDB - NoSQL database for user data and posts

Architecture Highlights

  • Implemented a double queue system to handle high traffic loads
  • Designed for horizontal scaling to support growing user base
  • Used serverless architecture for cost optimization
  • Applied software architecture best practices and design patterns
